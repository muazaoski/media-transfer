How to Use

Quick Start

To start the server instantly, just double-click start-server.bat in the main folder.

The batch file will automatically install dependencies and start the server for you.

Steps

  1. Connect to WiFi - Make sure your device and PC are on the same network
  2. Open Browser - Access this page from your mobile device
  3. Select Files - Click "Choose Files" and select photos or videos
  4. Upload - Click "Upload Files" to transfer them to your PC
  5. Done - Files are saved in /uploads folder
